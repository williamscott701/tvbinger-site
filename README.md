# TVBinger — Landing Page

The public marketing site for **TVBinger**, a free Chrome extension that overlays
IMDb, Rotten Tomatoes and Metacritic ratings on Netflix, Prime Video, Disney+, HBO
Max and JioHotstar — plus skip-intro, on-screen seek controls, and synced watch
parties with voice & video chat.

- **Live site:** https://williamscott701.github.io/tvbinger-site/
- **Get the extension:** https://chromewebstore.google.com/detail/ojnaeameimeelpfnmfkfkcpnknohpkmb

This repo holds only the static landing page (HTML + screenshots), served via
GitHub Pages. It's built and published from the extension's private source
repository — there's no application code here.
